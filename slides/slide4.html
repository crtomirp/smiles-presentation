Step-by-Step Construction

Example: Isobutane

1. Pick a starting atom and begin a chain.CC
2. At the branch point, open parentheses.CC(
3. Add the branched atom.CC(C
4. Close parentheses and continue the main chain.CC(C)C
5. Done!

Narration: Let's construct the SMILES for isobutane. We start by tracing a continuous path of atoms. First, we write 'C-C' for the first two carbons. The second carbon has a branch, so we open a parenthesis and add the methyl group inside it, giving us 'C-C, parenthesis, C'. To return to the main path, we close the parenthesis. Finally, we add the last carbon in our main chain. The final string is 'C-C, parenthesis, C, parenthesis, C'.
