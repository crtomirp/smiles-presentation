What is SMILES?

Definition

Simplified Molecular Input Line Entry System.

Purpose

A compact, human-readable, and machine-readable string representation of a molecule's structure.

Ethanol becomes the SMILES string:

CCO

Narration: SMILES is an acronym for Simplified Molecular Input Line Entry System. Its purpose is to encode the two-dimensional structure of a molecule into a compact string of text. This text format is easy for both humans and computers to read, making it essential for chemical databases and computational software. For example, the ethanol molecule, which has a two-carbon chain and an oxygen, is simply represented by the three-letter string 'C-C-O'.
