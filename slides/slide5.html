Canonical vs. Non-Canonical

Canonical SMILES

The single, unique SMILES string generated by a specific algorithm.

Essential for database lookups and ensuring that one molecule has only one definitive representation.

Non-Canonical SMILES

Multiple valid strings can represent the same molecule.

Ethanol can be: CCO OCC

Narration: Because you can start tracing a molecule from different atoms, a single structure can have multiple valid SMILES strings. As you see with ethanol, both 'C-C-O' and 'O-C-C' are correct. These are called non-canonical SMILES. To avoid confusion in databases, algorithms generate one standardized, unique string called the canonical SMILES. Think of it as the molecule's official name, while non-canonical forms are like nicknames.
