SMILES Building Blocks

A SMILES string is a line notation that unambiguously describes a molecule's structure using a small set of simple rules and characters. Below are the fundamental components you will use to write any molecule.

Atoms

C, N, O, S, P, F, Cl, Br, I

Bonds

-=#:

Branches

( )

Rings

1, 2, ... 9, %10, %11...

Narration: The power of SMILES comes from its simplicity. It uses just four types of building blocks. Atoms are written using their element symbols. Bonds are shown with characters for double, triple, and aromatic, but the single bond is usually omitted to keep the string short. Any group of atoms that branches off the main chain is enclosed in parentheses. Finally, rings are indicated by placing a matching number on the two atoms that need to be connected to form the ring.
